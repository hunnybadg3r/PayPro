# MockPOS

## 패킷 구조

MockPOS에서 사용하는 패킷은 ASCII 제어 문자를 사용하여 필드를 구분합니다.

### 예시 패킷

\u0002PAYMENT_REQ\u001c0624-000001\u001c20240626143000\u001cHD001\u001cPOS001\u001cOP123\u001cGASOLINE\u001c50000.50\u001c1652.89\u001c30.25\u001cCREDIT_CARD\u001c1234567890123456\u000353\r\n

### 필드 설명

| 순서 | 필드 | 설명 | 예시 값 |
|------|------|------|---------|
| 1 | 시작 문자 | 패킷 시작을 나타냄 | \u0002 (STX) |
| 2 | 패킷 타입 | 요청 유형 | PAYMENT_REQ |
| 3 | 거래 ID | 고유 거래 식별자 | 0624-000001 |
| 4 | 타임스탬프 | 거래 시간 (YYYYMMDDHHmmss) | 20240626143000 |
| 5 | 지점 ID | 주유소 지점 식별자 | HD001 |
| 6 | POS 터미널 ID | POS 기기 식별자 | POS001 |
| 7 | 운영자 ID | 주유원 식별자 | OP123 |
| 8 | 연료 유형 | 주유한 연료 종류 | GASOLINE |
| 9 | 금액 | 결제 금액 (원) | 50000.50 |
| 10 | 단가 | 리터당 가격 (원) | 1652.89 |
| 11 | 부피 | 주유량 (리터) | 30.25 |
| 12 | 결제 방식 | 사용된 결제 수단 | CREDIT_CARD |
| 13 | 식별 번호 | 결제 카드(토큰) 번호 | 1234567890123456 |
| 14 | 종료 문자 | 패킷 종료를 나타냄 | \u0003 (ETX) |
| 15 | 개행 문자 | 패킷의 끝 | \r\n (CR LF) |

\* 필드 구분자: \u001c (FS, Field Separator)
